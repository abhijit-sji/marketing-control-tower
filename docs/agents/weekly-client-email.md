# Weekly Client Email

> **Last Updated:** 2026-01-15  
> **Status:** ✅ Active  
> **Slug:** `weekly-client-email`

## Overview

The Weekly Client Email agent generates professional weekly summary emails for clients based on project task activity and comments from ActiveCollab. It aggregates work completed during a specified week and creates clear, actionable summaries.

## Key Features

- **ActiveCollab Integration**: Fetches task comments directly from ActiveCollab API
- **Date Range Filtering**: Summarizes activity for specific Monday-Friday period
- **Project Grouping**: Organizes updates by project
- **AI Summarization**: GPT-4 generates professional, client-ready summaries
- **Comment Extraction**: Handles various ActiveCollab comment field formats

## Scope

- **Type**: Client
- **Required Role**: Manager or Super Admin
- **Access**: Admin Panel → AI Control → Weekly Client Email

## Data Sources

| Source | Table/API | Purpose |

## Workflow

### Primary Flow (with project_ids)

1. **Validate Request**: Ensure client_id, dates are provided
2. **Fetch Client Info**: Get client name from database
3. **Fetch Project Names**: Get project names from database
4. **For Each Project**:
   - Fetch all tasks from ActiveCollab API
   - For each task: Query comments via SQL endpoint
   - Filter comments within date range
   - Extract comment text using multiple field fallbacks
5. **Generate Summary**: Call GPT-4 with task/comment data
6. **Return Summary**: Markdown-formatted email content

### Fallback Flow (without project_ids)

1. Query `activecollab_task_data` for cached task data
2. Filter by `last_comment_date` within range
3. Fetch full comments from ActiveCollab API
4. Continue with summarization

### Comment Text Extraction

The agent handles various ActiveCollab comment formats:

```typescript
function extractCommentText(comment: any): string {
  const text = comment.body || 
               comment.comment || 
               comment.body_plain || 
               comment.body_formatted || 
               comment.html_body ||
               comment.text ||
               '';
  
  if (typeof text === 'string') return text.trim();
  if (typeof text === 'object') {
    return (text.text || text.content || text.body || '').toString().trim();
  }
  return '';
}
```

## ActiveCollab Integration

### API Endpoints Used

| Endpoint | Purpose |
|----------|---------|
| `/ac-get-all-tasks` | Fetch tasks for a project |
| `/ac_query_tasks.php` | SQL query for comments |

### Comment Query
```sql
SELECT * FROM comments 
WHERE parent_type = 'Task' 
AND parent_id = ${taskId}
```

## Best Practices

1. **Select Relevant Projects**: Focus on projects with activity
2. **Monday-Friday Range**: Standard work week for consistency
3. **Review Before Sending**: Always review AI summary before client delivery
4. **Add Personal Touch**: Consider adding personalized intro/outro

## Error Handling

- **No Projects Found**: Returns empty summary with message
- **No Comments in Range**: Returns message indicating no activity
- **API Errors**: Continues with other tasks/projects, logs errors
